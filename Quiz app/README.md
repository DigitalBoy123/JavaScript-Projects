## Features
- **Multiple-Choice Questions**: 7 thoughtfully crafted questions on web technologies.
- **Real-Time Scoring**: Automatically tracks correct answers and displays results.
- **Interactive UI**: Custom radio buttons with visual checkmarks for selections.
- **Alert System**: Prompts users to select an option before proceeding.
- **Result Summary**: Shows final score with an option to restart.
- **Responsive Design**: Clean layout with gradient backgrounds and smooth transitions.

## Technologies Used
- **HTML**: Structures the quiz interface, questions, and options.
- **CSS**: Provides a vibrant, responsive design with custom-styled elements.
- **JavaScript (ES6 Modules)**: Handles question loading, answer checking, and result display.
