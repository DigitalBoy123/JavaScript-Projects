# Calculator

Dive into effortless calculations with this sleek, modern web-based calculator! Crafted with a minimalist design and intuitive interface, it handles basic arithmetic operations like addition, subtraction, multiplication, and division with precision and style.

## Features
- **Basic Operations**: Perform addition (+), subtraction (-), multiplication (ร), and division (รท) seamlessly.
- **Smart Input**: Includes decimal points, clear (C), and delete functions for easy corrections.
- **Error Handling**: Displays "Error!" for invalid operations, ensuring a smooth user experience.
- **Responsive Design**: Optimized for various screen sizes with a neumorphic aesthetic.
- **Limit Management**: Caps input at 11 characters and formats long results for readability.

## Technologies Used
- **HTML**: Structures the calculator layout.
- **CSS**: Provides the stylish, dark-themed interface with smooth animations.
- **JavaScript**: Powers the interactive functionality and expression evaluation.

## How to Use
1. Clone or download the repository.
2. Open `index.html` in your preferred web browser.
3. Start calculating by clicking the buttons!
