# Rock Paper Scissors Game

Engage in the timeless battle of wits with this classic Rock Paper Scissors game! Face off against the computer in quick, strategic rounds, track your scores, and enjoy instant feedback on every move. Perfect for a fun break or sharpening your decision-making skills.

## Features
- **Interactive Gameplay**: Click on rock, paper, or scissors images to make your choice.
- **Random Computer Moves**: Computer selects randomly for fair competition.
- **Score Tracking**: Displays live scores for you and the computer.
- **Visual Feedback**: Color-coded messages for wins (green), losses (red), and draws (blue).
- **Responsive Design**: Adapts to mobile and desktop screens.
- **Hover Effects**: Images highlight on hover for better user experience.

## Technologies Used
- **HTML**: Structures the game layout, images, and score displays.
- **CSS**: Creates a sleek, dark-themed interface with smooth animations and media queries.
- **JavaScript**: Manages game logic, event handling, and score updates.
